Deliverable 01 · Active-state overlay

The pill that lives at the edge of attention

A 200×56 always-on-top pill, bottom-center, visible only while Sotto works. Each canvas paints a desktop then the pill on top at 90% opacity — the real translucency stack, no blur. State is color, so status reads peripherally without reading a word. Every state is shown on a dark and a light desktop to prove contrast.

Deliverable 02 · Motion

Motion that reassures, never demands

Everything here is 60fps-cheap and built from primitives egui already has: moving points, a phase offset, a lerp toward a target, an alpha ramp. Nothing spins fast, nothing bounces. The demos below are live.

Listening · waveform
A continuous wave whose amplitude tracks live mic level and whose phase drifts sideways. Tapered to zero at both ends so it never clips the pill.
movesphase drift + per-frame amplitude envelope amplitude±12 px peak · sin-window taper to 0 at ends enveloperetarget every 110 ms · lerp k = 0.12 / frame phase loop≈ 1260 ms · seamless stroke2.4 px line + 5 px @ 22% underlay (bloom)
Transcribing · traveling dot
The wave collapses to a flat dim track. A single amber dot with a short leading trail sweeps across it — indeterminate, calm, clearly "working."
movesdot + 22 px gradient trail, left → right easingease-in-out, both edges soft loop1150 ms / pass · repeats track2 px line, amber @ 30%
Polishing · dot + sparkle
Same traveling dot, warmer gold, plus small 4-point sparkles twinkling above and below the track. The sparkle is the whole tell that intelligence is at work.
movesdot (1250 ms) + spawned sparkles sparklespawn ~210 ms · scale 0→1→0 (sin) life760 ms each · size 4–6.5 px accentgold #F0C982 · spark #FCEBC6
Done · tick & dismiss
A cyan check strokes itself on, holds a beat, then the entire pill fades out and stops rendering. The blank you see in the loop is the pill genuinely gone.
drawpath length 0 → full over 260 ms, ease-out hold140 ms at full fadeopacity + track → 0 over 400 ms
Error · quiet notice
No shake, no flash, no red. The rose glyph and label simply appear, rest long enough to read, and fade. Failure should feel like a shrug, not an alarm.
appearfade in 160 ms dwell~2400 ms readable fade300 ms out
